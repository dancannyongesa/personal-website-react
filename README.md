# Personal Website

Built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion for animations, and next-themes for dark mode.

## Features
- Responsive design
- Dark/Light mode toggle
- Smooth animations
- Blog section with dynamic routes
- Simple Admin panel for managing content (demo)

## Getting Started

```bash
npm install
npm run dev
```

Deploy on Vercel with one click!
